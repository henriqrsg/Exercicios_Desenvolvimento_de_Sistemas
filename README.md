Este repositório contém meus exercícios de programação das aulas de Desenvolvimento de Sistemas, organizados por número e nível. O objetivo é praticar e registrar meu aprendizado em diferentes linguagens e conceitos.

## Conteúdo

- Exercícios de Html5
- Exercícios de PHP
- Exercícios de SQL
- Exercícios de lógica de programação
- Criação de sites e minisistemas
- Pequenos projetos e desafios práticos
