# Exercícios de Programação — Desenvolvimento de Sistemas

Repositório dedicado à prática contínua de programação, com foco em construção de lógica, desenvolvimento web e integração com banco de dados.
Os exercícios estão organizados por nível de complexidade e tecnologia, refletindo minha evolução prática ao longo do curso.

---

## Objetivo

Consolidar conhecimentos em desenvolvimento de sistemas através de prática estruturada, incluindo:

* Construção de interfaces web
* Manipulação de dados com PHP
* Integração com banco de dados (MySQL)
* Controle de sessão e autenticação
* Organização de código e boas práticas

---

## Tecnologias Utilizadas

* HTML5
* CSS3
* PHP
* MySQL
* XAMPP (ambiente local)

## Como Executar o Projeto

1. Instale o XAMPP
2. Inicie o Apache e MySQL
3. Coloque a pasta do projeto em:

```
C:\xampp\htdocs\
```

4. Acesse no navegador:

```
http://localhost/exercicio_de_times
```
